## 통계별 추천 검증 경로

### 농산물 생산비 / 소득
```
1차: KOSIS 국가통계포털 > 농림 > 농산물소득
2차: 농촌진흥청 농사로 > 농산물소득자료집
3차: 공공데이터포털 > "생산비조사" 다운로드
```

### 보조금/지원 정책
```
1차: 공공데이터포털 > "농림축산식품부" 검색
2차: 농림축산식품부 > 정책홍보 > 보도자료 (최신)
3차: 정부24 보조금24 > 농업 카테고리
4차: 국가법령정보센터 > 시행지침서 원문
```

### 스마트팜 기술/정책
```
1차: 농촌진흥청 > 정책홍보 > 보도자료
2차: 농촌진흥청 농업기술 데이터 플랫폼
3차: 한국농촌경제연구원 > 이슈+ (정책 분석)
```

### 농가 통계 추이
```
1차: KOSIS 국가통계포털 > 농림 (시계열 데이터)
2차: KOSIS 100대지표 (주요 지표 요약)
3차: 농림축산식품부 통계연보 (년도별 종합)
```

## 자주 검증하는 항목별 자료원

### "정부가 스마트팜에 얼마나 지원하는가?"
- **검증 단계**:
  1. 농림축산식품부 정책 공고 검색
  2. 공공데이터포털에서 사업별 예산 확인
  3. 농촌진흥청 국고보조금 공개 자료 확인
  4. KREI 정책분석 리포트 참고

### "농가 소득은 얼마인가?"
- **검증 단계**:
  1. KOSIS > 농산물소득 검색
  2. 농사로 > 농산물소득자료집 다운로드
  3. 공공데이터포털 > 농산물소득조사 원본
  4. KREI 농업전망 참고

### "스마트팜 기술은 어떤 수준인가?"
- **검증 단계**:
  1. 농촌진흥청 보도자료 (최신 개발)
  2. 농업기술 데이터 플랫폼 (기술 사양)
  3. 연구보고서 (기술 평가)
