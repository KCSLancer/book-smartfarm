# 3장·4장 참고자료 페이지 초안

이 문서는 작업용 초안이다. 독자에게 보여줄 정리본은 `reference/chapter03_04_reference_page.md`에 따로 작성한다.

이 문서는 3장과 4장에서 별도의 참고자료 페이지로 옮기기 쉬운 형태로 정리한 초안이다.

- 원칙: 아직 공식 연도와 자료명이 확정되지 않은 문장은 본문을 먼저 일반화하고, 근거는 기관명과 연도가 확인된 뒤 참고자료 페이지에 정리한다.
- 표기 목표: 장별 참고자료 페이지
- 우선순위: 3장 제도 문장 -> 4장 비용·시설 문장

## 3장

### 3.1 정책자금 구조

- 현재 문장 성격: 후계농·창업농 자금이 보조금이 아니라 보증부 대출 구조라는 설명
- 참고자료 페이지 기재 초안:
  - 농림축산식품부. 후계농업경영인 육성사업 시행지침. 연도.
  - 농림축산식품부. 청년후계농 영농정착지원사업 시행지침. 연도.
  - 농림수산업자신용보증기금. 보증제도 안내. 연도.
- 확인할 자료:
  - 후계농업경영인 육성사업 시행지침
  - 청년후계농 영농정착지원사업 시행지침
  - 농림수산업자신용보증기금 제도 안내
- 검색어:
  - 후계농업경영인 육성사업 시행지침 pdf
  - 청년후계농 영농정착지원사업 시행지침 pdf
  - 농신보 보증제도 안내

### 3.1 영농정착지원금 성격

- 현재 문장 성격: 생활안정 지원인지, 운영자금인지 구분하는 설명
- 참고자료 페이지 기재 초안:
  - 농림축산식품부. 청년후계농 영농정착지원사업 시행지침. 연도.
- 확인할 자료:
  - 청년후계농 영농정착지원사업 시행지침

### 3.3 농업인의 신용·매출 기록 한계

- 현재 문장 성격: 농업의 면세 구조와 소득 증빙의 어려움 설명
- 참고자료 페이지 기재 초안:
  - 국세청. 농업 관련 부가가치세 면세 안내. 연도.
  - 국가법령정보센터. 부가가치세법 및 관련 시행령. 연도.
- 확인할 자료:
  - 국세청 농업 관련 부가가치세 면세 안내
  - 국가법령정보센터 부가가치세법 관련 조항
- 메모:
  - 이 문장은 공식 자료에서 일반론만 확인하고, 경험적 해석은 현재 본문 서술을 유지하는 편이 안전함.

### 3.4 공익직불금 감액·처벌 구조

- 현재 문장 성격: 행정 감액과 부정수급 처벌 구조 설명
- 참고자료 페이지 기재 초안:
  - 국가법령정보센터. 농업·농촌 공익기능 증진 직접지불제도 관련 법령. 연도.
  - 농림축산식품부. 공익직불제 안내자료. 연도.
- 확인할 자료:
  - 농업·농촌 공익기능 증진 직접지불제도 운영 관련 법령
  - 농림축산식품부 공익직불제 안내자료
- 검색어:
  - 공익직불금 준수사항 감액 법령
  - 공익직불금 부정수급 제재

## 4장

### 4.2 미래산업 표현의 거리

- 현재 문장 성격: 기술 홍보와 사업 현실 사이의 간극 설명
- 참고자료 페이지 기재 초안:
  - 농촌진흥청. 곤충산업 또는 스마트농업 관련 홍보자료. 연도.
- 확인할 자료:
  - 농촌진흥청 곤충산업 또는 스마트농업 홍보 자료
- 메모:
  - 이 부분은 수치 인용보다 홍보 문구의 존재만 확인해도 충분함.

### 4.3 농지·시설 비용

- 현재 문장 성격: 농지와 시설이 개인에게 큰 초기 부담이라는 설명
- 참고자료 페이지 기재 초안:
  - 국토교통부. 공시지가 또는 농지 실거래 관련 자료. 연도.
  - 농촌진흥청. 시설원예·스마트팜 표준모델 자료. 연도.
  - 스마트팜 보급사업 공고문. 연도.
- 확인할 자료:
  - 국토교통부 공시지가 또는 농지 실거래 관련 자료
  - 농촌진흥청 시설원예·스마트팜 표준모델 자료
  - 스마트팜 보급사업 공고문

### 4.4 시설 유형별 차이

- 현재 문장 성격: 시설별 비용과 운영 난이도 차이 설명
- 참고자료 페이지 기재 초안:
  - 농촌진흥청. 스마트팜 진단 또는 시설원예 기술자료. 연도.
  - 관련 사업 공고의 시설 유형 설명서. 연도.
- 확인할 자료:
  - 농촌진흥청 스마트팜 진단 또는 시설원예 기술자료
  - 관련 사업 공고의 시설 유형 설명서

## 바로 다음 반영 순서

1. 3장 3.1과 3.4의 법령·시행지침 자료명을 먼저 확정한다.
2. 4장 4.3과 4.4의 비용·시설 자료를 확정한다.
3. 연도까지 확인되면 장별 참고자료 페이지에 기관명, 자료명, 연도 순으로 바로 정리한다.
4. 연도 확인이 안 되는 문장은 지금처럼 본문은 일반화한 표현을 유지하고, 참고자료 페이지에서도 후보 자료만 남긴다.
