# 참고자료 페이지 정리 대상 목록

## 목적

이 문서는 본문에서 `경험 진술`이 아니라 `검증 가능한 사실 진술`에 해당하는 문장을 장별로 정리한 작업 목록이다.

- 원칙: 숫자, 제도, 법적 지위, 시장 구조, 유통 관행, 가격 구조는 별도의 참고자료 페이지에서 장별로 정리한다.
- 제외: 저자 개인 경험, 현장 체감, 판단, 해석, 정서적 반응은 출처 대상이 아니다.
- 우선순위:
  - `A`: 출처 없으면 신뢰도에 직접 타격이 큰 문장
  - `B`: 출처가 있으면 좋지만, 정성 표현으로도 버틸 수 있는 문장
  - `C`: 가능하면 별도 참고자료 페이지의 간단한 참고 목록 수준으로만 보강해도 되는 문장

## 우선 작업 순서

1. 3장, 4장, 10장, 11장, 12장의 `A` 항목부터 처리한다.
2. 공식 수치가 확인되면 장별 참고자료 페이지에 기관명, 자료명, 연도 순으로 정리한다.
3. 공식 수치가 확인되지 않으면 숫자를 더 줄이거나 정성 표현으로 바꾼다.
4. `B` 항목은 정부 자료가 없으면 KREI, 공공데이터, 보도자료, 시장 자료 순으로 보강한다.
5. `C` 항목은 무리하게 본문과 연결하지 말고 참고자료 페이지의 짧은 참고 목록 수준으로 처리한다.
