## 해결된 이슈

### 1. 8장 수정 필요 (RESOLVED)

유저가 "내가 쓴 내용과 다르다"고 지적. 가족관계 악화, 자살 위험 등 note.md에 없는 감성적 부풀림이 원인.
- **해결**: part3.md 8장에서 감성적 부풀림 전부 삭제, "대체 인력 부재 시 작업이 멈춘다"로 재구성 (4절, 12p)

### 2. 10장 수정 필요 (RESOLVED)

유저가 "내가 언급한 내용과 다르다"고 지적. "연간 인건비 5~10억원", "완전손실 시나리오" 등 note.md에 없는 과대 추정이 원인.
- **해결**: part3.md 10장에서 과대 추정치/시나리오 전부 삭제, "생산비 파악의 어려움과 기록 부재"로 재구성 (4절, 12~13p)

### 3. 5부(13-16장) 축소 (RESOLVED)

유저: "5부의 내용은 가볍게 언급할 내용이 아니며, 책으로 옮길 내용이 아님. 간단히 언급하고 마무리 하는 것이 필요"
- **해결**: 4장/17절/66p → 1장(13장)/6절/15p로 축소. note.md 라인40-46의 6개 bullet point만 반영. Momus 4차 리뷰 GO

### 4. 전체 outline 비난/가치판단 표현 점검 (RESOLVED)

유저: "3장에서 정부는 책임을 회피한다는 말은 한적없음. 전체적으로 사실에 부함하지 않는 내용이 있는지와 누구의 잘못/탓/책임을 의미하는 내용들이 있는지 점검해줘"
- **해결**: part1.md 3장 8건, part2.md 1건, toc.md 4건 수정 완료. grep 검증 0건 잔존 확인.

### 5. Phase 20 전체 비판적 검토 + 사실검증 (RESOLVED)

유저: "목차와 내용을 에이전트별로 비판적인 입장에서 보고 의견을 내줘. 특히 지원금의 99%+1%는 너무 일반화한 것 같은데 검토해줘"
- 3개 에이전트 병렬 조사 + Oracle 종합 비판 리뷰 → 12건 수정 권고 + 누락 3건(N1/N2/N3) 도출
- **해결**: toc.md 6건, part1.md 8건, part2.md 4건, part3.md 4건(7.4/8.4 재구성, 10.4 단편화, 10.1 임차), part5.md 3건(센서역설 해소, 임차 추가, 법인 유형 비교) 모두 수정 완료
- grep 최종 검증 통과, toc.md ↔ part*.md 동기화 완료

### 6. 제4부(11-12장) 인삼 특화 재작성 (RESOLVED)

유저: "4부의 내용과 내가 느낀 내용은 많이 다름" — AI 일반론 제거, 유저 경험 기반 인삼 특화 콘텐츠로 전면 교체
- 유저가 상세 경험 제공: 출하 프로세스, 선별(100% 수작업, 20여 등급), 판매 채널(계약출하 90% 독점 + 시장출하 과점 + 온라인/직거래), 5종 농산물, 노쇼사기, 거래 단위(차 750g)
- 인삼 시장 사실검증: 금산인삼시장 70-80% 유통, 경매 아닌 중개거래 확인
- Oracle 컨설팅: 11장=물리적 프로세스, 12장=경제적 구조 분리 + 스마트팜 동기 → 프롤로그 이동 권고
- **해결**: part4.md 11-12장 전면 재작성 + part1.md 프롤로그 2절 보강(가업/계약재배/종묘삼/경제적 돌파구) + toc.md 동기화 + grep 최종 검증 통과
